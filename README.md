# zhihu_user
通过一个种子用户，获取种子用户的关注者，再获取种子用户关注者的关注者，进而爬取所有知乎用户。目前爬取了3万多条用户信息（昵称，回答数，粉丝数，性别），利用pandas去重得1万多条数据，筛选粉丝数大于1000的用户得到7000多条。后续数据分析工作暂待更新。
